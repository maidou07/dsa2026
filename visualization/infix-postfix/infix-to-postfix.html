Algorithm Steps

  1. Initialize empty stack and empty output list
  2. Read tokens from infix expression left to right
  3. If operand (letter/number), add to output
  4. If operator, pop from stack while stack has operators with higher or equal precedence and add to output, then push current operator
  5. If '(' push to stack
  6. If ')' pop from stack to output until '(' is found
  7. After reading all tokens, pop remaining operators from stack to output
